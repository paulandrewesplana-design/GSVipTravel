# GS VIP Travel — Single-File GitHub Pages Build

This version is fully self-contained. All artwork, styling, and JavaScript are embedded inside `index.html`.

## Upload
1. In the GitHub repository, open the existing `index.html`.
2. Choose **Edit** or delete/replace it.
3. Upload this new `index.html` to the repository root.
4. Commit the change.
5. GitHub Pages should update within a few minutes.

No `images` folder is required for this version.
